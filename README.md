# BallWithinBound

一个在旋转多边形边界内弹跳的球的动态物理模拟，具有实时音频反应视觉效果和可定制的物理参数。

## 环境要求

- Python 3.7+
- pygame-ce >= 2.5.7
- numpy >= 1.21.0
## 安装步骤

1. 克隆仓库：
   ```bash
   git clone https://github.com/But1236/BallWithinBound.git
   cd BallWithinBound
   ```

2. 安装所需依赖：
   ```bash
   pip install -r requirements.txt
   ```

3. 确保项目目录中有必要的音频文件：
   - `coin-collect.mp3`
   - `toast-glass.mp3`
   - `wood-crack.mp3`
   - `starwar.mp3` (用于音乐模式)

## 使用方法

运行主应用程序：
```bash
python main.py
```

### 控制说明

#### 键盘控制：
- `R` - 使用相同种子重新初始化球
- `N` - 使用新随机种子重新初始化球

#### 鼠标控制：
- **滑块**：
  - 第一个滑块：调整加速度（重力）
  - 第二个滑块：调整碰撞系数（弹性）
  - 第三个滑块：调整轨迹持续时间
  
- **按钮**：
  - **EDGE**：打开/关闭 碰撞时自动增加边数
  - **GLOW**：打开/关闭霓虹发光效果
  - **TRAIL**：打开/关闭运动轨迹
  - **SOUND**：切换碰撞声音选项（关闭、声音1、声音2、声音3）
  - **MUSIC**：打开/关闭音乐盒模式

### 音乐盒模式

启用音乐模式时：
- **小球会踩着鼓点发生碰撞**
- 重力设置为零
- 球的速度会根据音乐节拍进行调整
- 多边形边数迅速增加到128条边
- 自动启用轨迹可视化
- 粒子效果得到增强

## 工作原理

模拟实时计算物理：
1. **多边形旋转**：多边形以恒定速度连续旋转
2. **球体物理**：球体受重力影响，根据速度矢量移动
3. **碰撞检测**：复杂的算法检测与多边形边缘和顶点的碰撞
4. **碰撞响应**：物理计算确定反弹角度和速度变化
5. **节拍检测**：在音乐模式下，系统检测节拍并同步球体运动

## 自定义设置

您可以在 `main.py` 中修改几个参数：
- 窗口大小 (WIDTH, HEIGHT)
- 多边形属性 (半径, 旋转速度)
- 球体属性 (半径, 初始位置)
- 物理参数 (默认加速度, 碰撞系数)
- 视觉效果 (颜色, 轨迹持续时间)
- 音频设置 (音量阈值, 节拍检测参数)

## 音频文件说明

项目包含以下四个音频文件，每个都有特定的用途：

- `coin-collect.mp3` - 声音效果1 (S1)，当启用声音效果并发生碰撞时播放
- `toast-glass.mp3` - 声音效果2 (S2)，当启用声音效果并发生碰撞时播放
- `wood-crack.mp3` - 声音效果3 (S3)，当启用声音效果并发生碰撞时播放
- `starwar.mp3` - 音乐模式背景音乐，在音乐模式下循环播放

用户可以通过 SOUND 按钮在不同声音效果之间切换，或完全关闭声音。

## 功能特点
- **动态多边形物理**：观察球在可调节边数的旋转多边形内弹跳
- **实时物理控制**：动态调整重力、碰撞弹性和其他物理属性
- **音频集成**：体验与音乐播放同步的视觉效果和节拍检测
- **视觉效果**：
  - 多边形的霓虹发光轮廓
  - 碰撞时的粒子爆炸效果
  - 运动轨迹可视化
  - 动画UI元素
- **交互控制**：直观的滑块和按钮，用于实时参数调整
- **多种音效**：三种不同的碰撞声音，可以切换
- **音乐模式**：特殊模式，使物理效果与音乐节拍同步
